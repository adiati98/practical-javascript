-----------

ğŸ”… .forEach() is to run every single item in array. It's shorter than for loop.
    It's handling the for loop for us.


ğŸ”… There are several ways to do .forEach():

   âœ  Create a variable with an array as value.

            var students = ['Jonathan', 'Jenny', 'Jerry'];

            
    1.  Create function and insert function name in .forEach() to call it.

            function logName(name) {
               console.log(name);
            }

            students.forEach(logName);

            students.forEach(function logName(name) {
               console.log(name);
            })

            students.forEach(function(name) {
               console.log(name);
            })
